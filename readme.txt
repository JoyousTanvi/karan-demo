*install following :

1. Node  - https://nodejs.org/en/download/
2. Git - http://git-scm.com/
3. Bower - open cmd promp and run : npm install -g bower

Restart machine after installations.

--------------------------------

*open command prompt and run the following commands

1. in project folder

$/demo> bower install

2. in server folder

$/server> npm install express


--------------------

To run the server

$/server> node server.js

To run the demo, open explorer and type url : localhost:8181/
